# Water Distortion #

<!-- ![](./public/preview.gif) -->

## Download ##

`git clone https://github.com/UstymUkhman/water-distortion.git`

`cd water-distortion`

`pnpm i`

## Develop ##

`pnpm start`

## Build ##

`pnpm build`

## Preview ##

`pnpm serve`
